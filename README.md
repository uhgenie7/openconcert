## 주요 특징

- 로그인, 회원가입, 로그아웃 기능
- 권한에 따른 글쓰기 기능
- 공지사항, QNA, 공연 검색 기능
- 게시판 다음글, 이전글 기능
- 게시판 글 번호 페이징 기능
- MariaDB와 JSON 연동한 비동기 기능 (perform page)
